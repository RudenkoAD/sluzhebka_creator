# Генератор служебок

Сайт для удобной генерации .docx документов

## Как запустить

### frontend

```bash
npm install
npm run build
```

Если вы собираетесь разрабатывать фронтенд, то используйте watch вместо build, он автоматически пересобирает страницу при изменениях
